# simple-port-scanner
# Простой многопоточный сканер открытых портов
Образовательный проект для демонстрации базового reconnaissance в кибербезопасности.
Важно: Используйте ТОЛЬКО на своих машинах, локальной сети или на тестовых серверах (например, scanme.nmap.org). Сканирование чужих хостов без разрешения незаконно.

Возможности
- Сканирование топ-30 популярных портов или заданного диапазона
- Многопоточность (быстрое сканирование)
- Определение известных сервисов (SSH, HTTP, HTTPS и др.)
- Сохранение результатов в txt-файл
- Таймер выполнения

Требования
- Python 3.6+
- Никаких дополнительных библиотек (только стандартная)
